# Проект «Cinemaddict»

## О проекте
SPA для просмотра фильмов с комментариями

## Что реализовано
- фильтрация
- сортировка
- комментарии (CRUD)
- popup с деталями
- MVP архитектура

## Архитектура
- Presenter / View / Model
- Observer pattern

## Мой вклад / чему научился
- работа с асинхронностью
- управление состоянием
- декомпозиция UI
